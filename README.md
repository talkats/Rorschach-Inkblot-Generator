## ✨ What is this?

An **interactive Rorschach Inkblot Generator** that creates unique symmetrical inkblots in your browser — just like the classic psychological test. Share your creations and see what others imagine!

## 🎮 Features

- 🎨 **Generate** unique symmetrical inkblots with one click
- 🔮 **Explore** creative personality-inspired interpretations
- 💾 **Export** your creations as **PNG** or **SVG**
- ⚡ **Zero dependencies** — pure HTML, CSS & JavaScript
- 📱 **Works on any device** — no install needed
